# Blog Draft Framework

Use this reference when the user wants the final article, not only the research briefing.

## Required Inputs

Prefer to confirm these before drafting:
- selected headline option
- target audience
- core thesis
- desired length or depth
- tone or stance
- optional region focus
- optional CTA, SEO keywords, or publication format

If the request lacks enough detail to make the article feel distinct, ask for the missing pieces before drafting.

## Drafting Principle

The article must be original synthesis, not a rewritten version of the source or the report.

Use the report as:
- evidence base
- thesis source
- tension map
- source of caveats

Do not use it as a sentence-by-sentence template.
Do not mention the source medium, transcript, or report in the article body.
Reserve source attribution for the source list and any explicit editorial notes outside the draft.

## Article Shape

Adapt the structure to the chosen headline and audience, but usually build:
1. Hook
2. Context
3. Core argument
4. Evidence and examples
5. Why it matters
6. Caveats or counterpoint
7. Practical takeaway
8. Closing line that matches the selected angle

## Style Rules

- Keep the article clearly rooted in the chosen thesis.
- Reframe the evidence into a fresh editorial arc.
- Avoid repeating the report's section titles as article headings.
- Avoid phrases like "the video says", "the source says", or "in the transcript".
- Prefer clear transitions over source chronology.
- Write in a voice that fits the target audience, not the source.
- Use plain language when possible, but keep the argument precise.
- Do not overquote the source; paraphrase and synthesize.

## Quality Checks

Before finishing, check:
- Does this read like a new article rather than a report copy?
- Does every section support the chosen thesis?
- Does the headline choice actually shape the narrative?
- Is the article understandable to the target audience?
- Are all important factual claims traceable to the report or sources?

## When to Ask for More Information

Ask for more context if the user did not specify:
- audience sophistication
- article length
- whether the piece should be explanatory, analytical, or opinion-led
- how strong the contrarian angle should be
- whether LATAM relevance should be explicit
